About the Project

This AI-powered resume analyzer automatically parses resumes, extracts structured information, and evaluates how well each resume matches a job description. It's built using deep learning and natural language processing to improve recruitment efficiency.

Developed with TensorFlow and integrated into a streamlined pipeline for accurate candidate-job matching.

Information Extracted

  • Job Title
  • Candidate Highlights
  • Direct Skill Matches
  • Related Skills
  • Experience Alignment
  • Keyword Frequency Score
  • Overall Matching Score

Project Description

This platform provides a powerful backend for resume parsing and intelligent job matching. Employers and job seekers alike can benefit from the intuitive analysis and recommendations generated by the AI engine. Built with scalable components, the model supports continuous improvement and integration with job portals.
